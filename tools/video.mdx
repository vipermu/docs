The Video Tool lets you generate AI-driven video clips from text prompts, images, or start/end frames. The menu changes slightly depending on which model you select.

## Main Components

<CardGroup cols={2}>
  <Card title="Prompt Box" icon="pencil">
    Describe the video you want to create
  </Card>
  
  <Card title="Prompt Settings" icon="sliders">
    Adjust model, resolution, aspect ratio, and style
  </Card>
  
  <Card title="Start Frame" icon="image">
    Upload an image to define the first frame of the video
  </Card>
  
  <Card title="End Frame" icon="images">
    (Only for supported models) upload an image to set the final frame
  </Card>
  
  <Card title="Styles" icon="palette">
    Apply a preset, community, or custom style
  </Card>
  
  <Card title="Asset Manager" icon="folder">
    View and organize your generated videos (top panel)
  </Card>
  
  <Card title="Generate Button" icon="play">
    Run the prompt and create video clips
  </Card>
  
  <Card title="Download Options" icon="download">
    Save your video
  </Card>
  
  <Card title="Send to Tools" icon="arrow-right">
    Forward a video into Enhancer, Restyle, or Motion Transfer
  </Card>
</CardGroup>

## Detailed Breakdown

### Prompt Box

<Steps>
  <Step title="Write Your Prompt">
    Write your prompt to describe the scene or motion.
  </Step>
  
  <Step title="Add Start Frame">
    Combine text with a start frame to anchor the composition.
  </Step>
  
  <Step title="Add End Frame (Optional)">
    Some models support adding an end frame to smoothly transition between two visuals.
  </Step>
</Steps>

### Prompt Settings

<AccordionGroup>
  <Accordion title="Resolution">
    Starts at 720p, higher options depend on the model.
  </Accordion>
  
  <Accordion title="Upscale">
    Final outputs can be enhanced up to 8K.
  </Accordion>
  
  <Accordion title="Format">
    Choose aspect ratio (landscape 16:9, portrait 9:16, square 1:1, etc.).
  </Accordion>
  
  <Accordion title="Style Intensity">
    Adjust how much influence the selected style has.
  </Accordion>
  
  <Accordion title="Model Selection">
    Pick from the list of video models (Kling, Veo, Seedance, Runway, etc.).
  </Accordion>
</AccordionGroup>

## Downloads & Actions

When you select a generated video, additional options appear:

<AccordionGroup>
  <Accordion title="Enhancement Actions">
    - **Upscale** → Improve resolution (up to 8K)
  </Accordion>
  
  <Accordion title="File Actions">
    - **Download** → Save the video locally
    - **Retry** → Re-run the prompt
    - **Reuse parameters** → Start a new clip with the same settings
  </Accordion>
  
  <Accordion title="Sharing Actions">
    - **Share parameters** → Share prompt + settings
  </Accordion>
  
  <Accordion title="Organization Actions">
    - **Delete** → Remove the video
    - **Like / Dislike / Favorite** → Organize and rate your results
  </Accordion>
</AccordionGroup>

## Available Models

Krea Video integrates a wide range of state-of-the-art models. Each differs in generation time, resolution, motion quality, and pricing:

### High-Quality Motion Models

<AccordionGroup>
  <Accordion title="WAN 2.1 / WAN 2.2">
    High-quality motion with strong prompt adherence.
    
    **Speed:** Medium | **Cost:** Medium
    
    [Learn more →](/ai-models/wan-video)
  </Accordion>
  
  <Accordion title="Kling 1.0 / 1.6 / 2.0 / 2.1">
    Cinematic, detailed, strong motion realism.
    
    **Speed:** Medium to slow | **Cost:** High
    
    [Learn more →](/ai-models/kling)
  </Accordion>
</AccordionGroup>

### Google's Cinematic Models

<AccordionGroup>
  <Accordion title="Veo 2 / Veo 3 / Veo 3 Fast">

</AccordionGroup>

## Model Selection Guide

<Info>
**Choosing the Right Model:**

- **For speed:** Seedance Lite, Hailuo, Pika 2.2
- **For quality:** Kling 2.1, Veo 3, Luma Ray 2
- **For experimentation:** 01-Live, Hailuo 2.0
- **For cinematic results:** Runway Gen 4, Veo 3, Kling series
- **For balanced results:** Hunyuan, WAN 2.2
</Info>
